# /cm-concept — 컨셉 결정

## 실행 조건
- 02_planning FEEDBACK.md에 ✅가 있어야 한다

## 수행 절차

### 1. 기획 승인 확인
02_planning/FEEDBACK.md를 읽어서 ✅ 승인 상태인지 확인.

### 2. 컨셉 결정 안내

```
🎨 컨셉 결정 단계입니다.

아래 파일을 작성해주세요:
👉 03_concept/CONCEPT-DECISION.md

작성 항목:
  - 제품 방향성: 이 제품이 추구하는 핵심 가치
  - 핵심 경험: 사용자가 느껴야 할 핵심 감정/경험
  - 톤앤매너: 제품의 성격 (전문적/친근한/미니멀 등)
  - 디자인 지향점: 참고할 제품이나 디자인 스타일

✅ 작성 완료 후 "컨셉 작성 완료" 라고 입력해주세요.
```

### 3. 작성 완료 처리
"컨셉 작성 완료" 입력 시:
1. 03_concept/CONCEPT-DECISION.md 파일 존재 확인
2. DASHBOARD.md에서 03단계를 `✅ 완료`로 변경

```
✅ 컨셉 결정이 완료되었습니다!

👉 다음 단계: /cm-tech 를 실행하여 기술 결정을 진행하세요.
```
